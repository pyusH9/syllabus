Chapter 4: Application Package

Application packages are software tools that help users perform tasks, from office work to multimedia creation.

Common Application Packages:

  • Microsoft Office: Includes Word, Excel, PowerPoint, etc.
  • Adobe Suite: Photoshop, Illustrator, and other design tools.
  • Web Browsers: Chrome, Firefox, Safari.
