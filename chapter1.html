Chapter 1: Computer System

A computer system is a combination of hardware and software that works together to perform various tasks. It includes components like the CPU, memory, input/output devices, and storage systems. Understanding how these components work is essential for becoming proficient in computer science.

1.1 Introduction of Computer

1.1.1 Definition, Characteristics and Application of Computer

A computer is an electronic device derived from the Latin word "computere," which means to calculate. It processes raw data into meaningful information through a sequence of operations, such as input, process, storage, and output. Some key characteristics include speed, accuracy, storage capacity, and versatility.

Applications of computers are widespread, including in education, business, communication, scientific research, engineering, and many more fields.

1.1.2 Evolution of Computer Technology

Computing technology has evolved significantly over the years, from mechanical devices like the abacus to modern electronic computers. Early computers were large and slow, using vacuum tubes. With the advent of transistors, microprocessors, and integrated circuits, computers became faster, smaller, and more affordable, paving the way for personal computers and mobile devices.

1.1.3 Measurement Unit of Processing Speed and Storage Unit

Processing speed is measured in Hertz (Hz), with modern processors operating in gigahertz (GHz). Storage capacity is typically measured in bytes (B), with common units being kilobytes (KB), megabytes (MB), gigabytes (GB), and terabytes (TB).

1.1.4 Super, Mainframe, Mini and Microcomputers

Computers come in various sizes and capabilities:

  • Supercomputers: Extremely powerful computers used for complex computations, like weather forecasting and scientific simulations.
  • Mainframe Computers: Large, high-performance systems used by organizations for bulk data processing.
  • Minicomputers: Smaller than mainframes but still used in medium-sized organizations for business and scientific applications.
  • Microcomputers: Personal computers, including desktops, laptops, and mobile devices.

1.1.5 Mobile Computing and its Application

Mobile computing refers to the ability to use computing devices like smartphones and laptops on the go. Applications include navigation, communication, entertainment, and access to online services.

1.2 Computer System and I/O Devices

1.2.1 Concept of Computer Architecture and Organization

Computer architecture refers to the structure of the computer system, including its CPU, memory, and input/output devices. Organization refers to how these components interact and work together to perform tasks.

1.2.2 Components of Computer System

  • Input Unit: Devices like keyboards, mice, and scanners used to enter data into the computer.
  • Output Unit: Devices like monitors and printers that display or output processed data.
  • Processing Unit: The CPU, where all the computations and data processing occur.
  • Memory Unit: Stores data temporarily (RAM) or permanently (ROM, hard drives).
  • Storage: Devices used for long-term data storage, like hard drives and SSDs.

1.2.3 Microprocessor

The microprocessor is the brain of the computer, performing the majority of calculations. It is characterized by its clock speed (the number of cycles per second), word length (how much data it can process at once), and its components like the arithmetic logic unit (ALU), control unit (CU), and registers.

1.2.4 Bus System

A bus system is a communication pathway used to transfer data between components of the computer system. The three main types are:

  • Data Bus: Carries the actual data being transferred.
  • Address Bus: Carries the memory address to or from which data is being sent.
  • Control Bus: Carries control signals to manage the operations of the CPU.

1.2.5 Primary Memory

Primary memory is used for quick data access and includes:

  • RAM (Random Access Memory): Volatile memory used to store data that is currently being processed.
  • ROM (Read-Only Memory): Non-volatile memory used to store essential system instructions.
  • Cache: A small, fast memory used to store frequently accessed data for quick retrieval.
  • Buffer: A temporary storage area used to hold data while it's being transferred between devices.

1.2.6 Secondary Memory

Secondary memory provides long-term data storage and includes:

  • Magnetic Disks: Hard drives used for large-scale data storage.
  • Flash Memory: A type of non-volatile memory used in USB drives, SSDs, and mobile devices.
  • Optical Disks: CDs, DVDs, and Blu-ray discs used for data storage and media playback.
  • External Storage: Devices like external hard drives or cloud storage used for backup and additional storage.

1.2.7 Input Devices

Input devices allow the user to enter data into the computer. Examples include:

  • Keyboard: A device used to input text and commands.
  • Mouse: A pointing device used to interact with the graphical user interface.
  • Scanner: Used to digitize physical documents.
  • Touch Screen: A display that detects user touch, commonly used in smartphones and tablets.
  • Microphone: Used for recording audio.
  • Digital Camera: Captures still images or videos.

1.2.8 Output Devices

Output devices display or produce the results of computer processing. Examples include:

  • Monitor (LCD, LED): Displays text and images on the screen.
  • Printer (Dot Matrix, Inkjet, Laser): Prints documents and images.
  • Speakers: Output sound from the computer.

1.2.9 Hardware Interfaces

Hardware interfaces are used to connect peripheral devices to the computer:

  • Parallel Port: Used for connecting printers and other devices.
  • Serial Port: Used for connecting devices like modems and mice.
  • USB Ports: A versatile and common interface for connecting devices like keyboards, mice, and external storage.
  • HDMI: Used to connect computers to monitors, projectors, and televisions.
  • Expansion Slots: Allow for the addition of expansion cards like graphics cards or network cards.
