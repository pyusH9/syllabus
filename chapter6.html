Chapter 6: Web Technology I

Web technology includes the tools and techniques used to build and manage websites. This chapter introduces the basics of web development, focusing on HTML, CSS, and JavaScript.

What is HTML?

HTML (HyperText Markup Language) is the foundation of web pages. It defines the structure of web pages using tags. For example, the <h1> tag defines the main heading, and the <p> tag defines a paragraph.

What is CSS?

CSS (Cascading Style Sheets) is used to control the appearance and layout of web pages. It defines colors, fonts, spacing, and positioning of elements on a webpage.

Introduction to JavaScript

JavaScript is a scripting language that adds interactivity to websites. It is commonly used for creating dynamic content such as pop-up windows, form validation, and animations.
